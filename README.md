# Language Learning Application

A feature-rich web application for language learning, enabling users to create personal libraries, manage playlists, and track their practice sessions.

## 🚀 Overview

This application provides a comprehensive platform for language learners to organize their study materials into "playlists" and engage in practice sessions. It features a discovery module to add new content, a personalized dashboard for progress tracking, and secure user authentication.

## ✨ Key Features

- **Personal Library**: Create and manage your own collection of language items.
- **Custom Playlists**: Organize language items into categorized playlists.
- **Interactive Practice**: Start sessions to test your knowledge and track correctness over time.
- **Progress Dashboard**: Visualize your learning activity and session history.
- **Content Discovery**: Find and add new language items to your library.
- **Secure Authentication**: Built-in sign-up and sign-in functionality via Supabase.

## 🛠️ Technology Stack

- **Frontend**: React (with Hooks and Context)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Backend API & Database**: Supabase
- **Build Tool**: Vite

## 🏗️ Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables:
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
